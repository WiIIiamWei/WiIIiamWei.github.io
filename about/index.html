关于我

William Wei（男）

喜欢信息技术但主修地信的大三生 （谁告诉我地信会学 CS 的所有内容的？！）

平常喜欢写 Python 的各种玩意，现在主要写 API（尤其 FastAPI）。可惜 Python 的就业机会仍然太少了，正在另外寻找一门合适的手艺……

使用 Windows / Arch Linux（仅仅日用）

音游玩家，成分复杂，详见关于音游。同时非常欢迎各类音游吃找我玩！

自我定位为宅男，二次元社恐，成分极其复杂，啥都知道点但大部分是云，喜欢看 V 但是没有任何牌子，纯纯的切片势。

XP 集中在机娘（身体是机械的萌娘，而非驾驶或装备机械的萌娘）这一块，因此喜欢收集奇怪的机娘头像，所以每个平台昵称基本一样，但头像不同。

在学习之余探索自我，有些焦虑，现在选择以竞赛促进学习，希望一切都好吧。

同时长期欢迎有想法、想一起学些什么或者想一起打竞赛的联系我。我参加的比赛基本都是团队合作出项目的那种（例如挑战杯），如果你智力超群、会写算法、想打 ICPC 或者蓝桥杯那还是算了，我菜（

有话想和我讲？考虑到隐私问题，所有的联系方式现已转到 Gravatar。请点击头像下的 Gravatar 图标或是 lit.link 图标/文末的卡片来找我，或是在评论区留言！

关于这里

这里是 William Wei 的个人小站，会看心情更新一些内容……

此前使用 WordPress 搭建博客，现在换到了 Hugo，感觉变清爽了很多……在原站点（已经关闭）上的博文就不公布/同步了。说得好听点：有一股清涩感；难听点：都啥玩意（不忍直视.webp）

本博客使用以下技术构建：

  • 静态站点生成 - Hugo（使用 Stack 主题，详见页面底部）
  • 字体 - jsDelivr（提供 HarmonyOS Sans SC，意外地觉得这个字体很美观）、Google Font API（提供 Cascadia Mono）

咱正好注册了一些免费域名。为确保可用性，本博客使用以下构建方式构建：
